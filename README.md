## Descripción de las páginas

- **index.html**: Página principal con información general del sitio. Incluye estadísticas de proyectos, tareas y mensajes, además de una sección de novedades.

- **proyectos.html**: Página donde se muestran los proyectos realizados, incorporando filtros de búsqueda. Cada proyecto incluye imagen, descripción y acceso al detalle.

- **detalle.html**: Página que presenta el detalle individual de cada proyecto, incluyendo fecha de inicio, autores y una descripción extendida.

- **perfil.html**: Página que muestra información personal del usuario, incluyendo imagen, nombre y rol.
